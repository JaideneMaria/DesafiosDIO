HTML

O que é HTML?

HTML é a sigla para HyperText Markup Language(Linguagem de Marcação de Hipertexto). O HTML é linguagem de marcação padrão, ela é responsável por estruturar uma página web e seu conteúdo. Consiste em uma série de elementos que informam como o navegador deve exibir o seu conteúdo. Exemplos: temos elementos que podem deixar uma palavra em negrito, itálicos, especificar que determinada frase é um título ou um parágrafo, informar o que são links etc. HTML é uma linguagem de programação MARCAÇÃO!!

Partes de um elemento HTML

Tag de abertura: É o nome do elemento que fica dentro dos símbolos "<" ">". Mostra onde o elemento se inicia.

Tag de fechamento: É a mesma coisa que a tag de abertura, exceto que inclui uma barra antes do nome do elemento. Isso demonstra onde o elemento termina.

Conteúdo: É o texto do elemento.

Exemplos:
Tag de título: <h1>Olá Mundo</h1>
Tag de parágrafo: <p>Esta é um parágrafo</p>

Atributos de um elemento

Atributos contém informação extra sobre o elemento que você não quer que apareça no conteúdo real.
Um atributo sempre deve ter:

  1. O nome do atributo, seguido por um sinal de igual.
  2. Aspas de abertura e fechamento, envolvendo todo o valor do atributo.
  3. Um espaço entre ele e o nome do elemento (ou o atributo anterior, se o elemento já tiver um).

Exemplos:
Tag de título: <h1 id="intro">Olá Mundo</h1>
Tag de parágrafo: <p style="color:red">Esta é um parágrafo</p>

CSS

O que é CSS?

CSS é a sigla para Cascading Style Sheets(Folhas de Estilo em Cascata). Ele é a linguagem usada para definir o visual de uma página web. Enquanto o HTML organiza o conteúdo, o CSS é responsável por dar cor, tamanho, fontes, espaçamento e layout aos elementos da página.

Por que o CSS é importante?

Sem CSS, todas as páginas na web ficariam com o mesmo visual básico: fundo branco, textos pretos e tudo alinhado à esquerda. O CSS permite criar interfaces bonitas, organizadas e responsivas, que funcionam bem em diferentes tamanhos de tela.

Como o CSS funciona?

O CSS atua selecionando elementos HTML e aplicando estilos a eles. Você pode alterar desde coisas simples, como a cor de um texto, até estruturas mais complexas, como grids e animações.

No CSS, podemos usar unidades como px, em e rem. Por exemplo, definir font-size: 12px significa que a fonte terá 12 pixels, e visualmente o "px" pode ser representado como px2 em alguns materiais técnicos.

Principais formas de usar CSS:

  • CSS inline: aplicado direto dentro da tag HTML.
  • CSS interno: colocado dentro da tag <style> no próprio arquivo HTML.
  • CSS externo: escrito em um arquivo .css separado e linkado ao HTML.

Exemplos do que o CSS pode fazer:

  • Mudar cores e fontes
  • Criar botões estilizados
  • Posicionar elementos na tela
  • Criar layouts responsivos
  • Aplicar sombras, bordas e efeitos

JavaScript

O que é JavaScript?

JavaScript é a linguagem que traz vida e interatividade para as páginas web. Enquanto o HTML estrutura e o CSS estiliza, o JavaScript permite que o usuário interaja com a página, envie informações, clique em botões e veja mudanças acontecendo em tempo real.

Por que o JavaScript é tão usado?

Com ele, é possível atualizar partes da página sem recarregar tudo, validar formulários, criar menus dinâmicos, animações, jogos, sliders, entre outras funcionalidades.

Onde o JavaScript roda?

Ele é executado diretamente no navegador do usuário, o que o torna rápido e ideal para criar experiências dinâmicas. Também pode ser usado no back-end (com Node.js), mas no front-end ele é o rei da interatividade.

O que dá pra fazer com JavaScript no front-end:

  • Mostrar e esconder elementos
  • Validar formulários
  • Criar menus interativos
  • Manipular textos e imagens
  • Atualizar conteúdo automaticamente

O JavaScript começou com o padrão ECMAScript 3ª versão e evoluiu muito até os dias de hoje.

Exemplo simples de uso:

Exibir uma mensagem quando o usuário clica em um botão, mudar a cor de um elemento, ou atualizar a tela com base numa ação.
